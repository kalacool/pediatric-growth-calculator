# 兒童生長曲線與骨齡預測計算器

一個專為兒童生長發育評估設計的網頁應用程式，提供生長百分位數計算、骨齡預測及遺傳目標身高估算等功能。

## 功能特色

- **生長百分位數計算**：根據兒童的年齡、性別、身高和體重，計算在生長曲線上的百分位數位置
- **骨齡預測身高**：利用骨齡檢查結果預測最終成年身高
- **遺傳目標身高**：根據父母身高計算遺傳潛在身高
- **早產兒調整**：支援早產兒的生長曲線調整計算
- **雙胞胎支援**：針對雙胞胎的生長評估提供調整
- **防呆機制**：內建輸入驗證和錯誤處理

## 如何使用

1. 開啟 `index.html` 檔案在瀏覽器中
2. 填入兒童的基本資料：
   - 性別
   - 生日
   - 身高（公分）
   - 體重（公斤）
3. 選填進階資訊：
   - 出生體重和懷孕週數（用於早產調整）
   - 父母身高（計算遺傳目標）
   - 骨齡檢查結果（預測最終身高）
4. 點擊「計算結果」按鈕查看分析結果

## 技術架構

- **前端**：純HTML5 + CSS3 + JavaScript（無框架依賴）
- **響應式設計**：可依據手機或桌面瀏覽器寬度自動縮放

## 安裝與運行

無需安裝，直接在支援HTML5的瀏覽器中開啟 `index.html` 即可使用。

### 系統需求
- 支援HTML5的現代瀏覽器（Chrome、Firefox、Safari、Edge）
- JavaScript啟用
- 建議螢幕解析度：至少1024x768

## 資料來源

1. **0-5歲生長曲線**：國民健康署公布之「兒童生長曲線」
2. **5歲以上生長曲線**：依據陳偉德醫師及張美惠醫師2010年發表之研究成果制定
3. **骨齡預測公式**：基於醫學文獻和臨床研究資料
